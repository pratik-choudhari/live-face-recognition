# live-face-recognition
A webcam face recognition program built using python

Copy images of people to recognize inside `images/` folder.

__Important: keep filename as the name of person.__

## Getting started:

1. Set up dependencies for windows, [click here](https://github.com/ageitgey/face_recognition/issues/175); for linux [click here](https://www.learnopencv.com/install-dlib-on-ubuntu/)
2. `pip3 install -r requirements.txt`
3. `python3 app.py`
